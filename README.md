# DeskGuard
Webcam posture monitor that flags slouching before it becomes a habit. No wearables. Runs entirely in your browser — no video is ever recorded or stored.

## Privacy first
All pose estimation runs locally in your browser using MediaPipe. Only anonymized angle data is ever sent to the backend.

## Stack
- Frontend: React + Vite + MediaPipe Tasks Vision
- Backend: Flask + OpenAI-compatible API
